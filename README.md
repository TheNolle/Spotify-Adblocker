# Spotify-Adblocker

### Description:
Skip Spotify's ads for you before it plays  |  Part of the Spotify Pack

### How To Use:
- Install [Tampermonkey](https://chrome.google.com/webstore/detail/tampermonkey/dhdgffkkebhmkfjojejmpbldmpobfkfo)
- Install the script
- Supports _ES5_/_ES6_
